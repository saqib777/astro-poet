# Astro Poet — for the restless seekers

A poetry website built from the *My_All_Poems* collection — 50 pieces
in English and Hindi/Urdu, presented as a brand hub linking out to
Medium, Substack, and Wattpad.

## Structure

```
index.html          Hub page: hero, social links, poem index grid
poem.html            Single poem reading page (reads ?p=<slug> from the URL)
css/style.css        All styling — dark starfield theme matching the original PDF
js/poems-data.js     Generated poem data (titles, bodies, slugs) — do not hand-edit
js/site.js           Shared nav/footer + social link config
js/starfield.js       Ambient canvas starfield + comets, layered over the nebula
assets/nebula.svg    Painted nebula background artwork (generated, see below)
generate_nebula.py   Regenerates assets/nebula.svg with a new random seed/layout
poems-data.json      Source of truth for poem content — edit this, not the .js file
build_data.py        Regenerates js/poems-data.js from poems-data.json
```

## Regenerating the background art

`assets/nebula.svg` is generated by `generate_nebula.py` — hundreds of soft,
blurred circles layered to look like a painted nebula rather than a flat
gradient. To get a different layout/color mood, change the seed at the top
of the file (`random.seed(7)`) and re-run:

```
python3 generate_nebula.py
```

## Editing poems

1. Edit `poems-data.json` (title, lang, body — use `\n` for line breaks,
   blank line between stanzas).
2. Run `python3 build_data.py` to regenerate `js/poems-data.js`.
3. Commit both files.

## Previewing locally

```
python3 -m http.server 8000
```

Then open `http://localhost:8000/index.html`.

## Deploying to GitHub Pages

1. Create a new repository on GitHub (e.g. `astro-poet`).
2. From this folder:
   ```
   git init
   git add .
   git commit -m "Astro Poet site"
   git branch -M main
   git remote add origin https://github.com/<your-username>/astro-poet.git
   git push -u origin main
   ```
3. On GitHub: go to the repo's **Settings → Pages**.
4. Under "Build and deployment", set **Source** to "Deploy from a branch",
   branch `main`, folder `/ (root)`. Save.
5. GitHub will publish the site at:
   `https://<your-username>.github.io/astro-poet/`
   (takes 1–2 minutes on first deploy)

No build step is required at deploy time — this is a static site, so
GitHub Pages serves the files as-is.
